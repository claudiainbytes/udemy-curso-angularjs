Ejemplo: Alias de controladores

Sin Alias, utilizando propiedad $parent.nombre

{{ nombre }}
{{ nombre + ', ' + $parent.nombre }}

Con Alias, a cada controlador se declara un alias para evitar el uso de $parent.nombre. A partir del alias, podemos acceder al atributo declarado en cada controlador sin llegar a confusiones

{{ padre.nombre }}
{{ hijo.nombre + ', ' + padre.nombre }}
