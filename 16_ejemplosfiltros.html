Ejemplos con Filtros

  • Filtro uppercase: {{'buenos dias'|uppercase}}
  • Filtro lowercase: {{'HOLA MUNDO'|lowercase}}
  • Filtro limitTo: {{'buenos dias'|limitTo:6}}
  • Filtro en cadena (lowercase + limit ): {{'ESTAS SON LAS MAÑANITAS'|lowercase |limitTo:-6}}
  • Filtro currency (moneda): {{ 2000 | currency:'€' }} o {{ 2000 | currency }}
  • Filtro fecha fullDate: {{ hoy | date:'fullDate' }}
  • Filtro fecha shortDate: {{ hoy | date:'shortDate' }}
  • Filtro fecha año dos digitos: {{ hoy | date:'yy' }}
  • Filtro fecha año cuatro digitos: {{ hoy | date:'yyyy' }}
  • Filtro para separar informacion de arrays: {{ ['mercurio','venus','tierra','marte','jupiter','urano','neptuno','pluton'] | filter:'o' }}
  • Filtro arrays: {{ planetas | filter:'ra' }}
  • Filtro array de objetos: {{ personas | filter:{pais: 'Colombia'} }}
